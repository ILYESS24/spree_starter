# Instructions pour supprimer l'ancienne base de données

## Étape 1: Supprimer l'ancienne base de données

1. Allez sur [Render Dashboard](https://dashboard.render.com)
2. Cliquez sur la base de données **evershop-db**
3. Allez dans l'onglet **Settings** (Paramètres)
4. Faites défiler jusqu'en bas
5. Cliquez sur **Delete Database** (Supprimer la base de données)
6. Confirmez la suppression

⚠️ **Attention**: Cette action est irréversible. Toutes les données seront perdues.

## Étape 2: Une fois supprimée

Une fois la base de données supprimée, dites-moi et je créerai automatiquement:
- ✅ Une nouvelle base de données PostgreSQL pour Spree
- ✅ Configuration de toutes les variables d'environnement
- ✅ Connexion Redis
- ✅ Redéploiement automatique
